Vibedrive is an intelligent assistive driving system designed to improve road safety for deaf drivers. 
The system detects important road sounds such as vehicle horns and emergency sirens and converts them into visual alerts.

## 🚀 Live Demo
https://vibedrive-model-64e137.netlify.app/

## System Architecture

VibeDrive processes environmental sounds and converts them into visual alerts for deaf drivers.

```
Audio Input
    ↓
Audio Capture
    ↓
Feature Extraction (MFCC)
    ↓
Sound Classifier (Machine Learning Model)
    ↓
Direction Detection
    ↓
Visual Alert System
```

## Features
- AI-based sound classification
- Emergency vehicle detection
- Direction estimation using simulated microphones
- Color-coded alert system
- Modular simulation architecture

## Technologies Used
- Python
- NumPy
- Librosa
- Scikit-learn
- Machine Learning

## Project Structure
audio_capture – audio input handling  
audio_processing – signal processing and feature extraction  
sound_classifier – machine learning model for sound classification  
alert_system – visual alert generation  
direction_detector – simulated sound direction detection  
simulator – full system pipeline  

## Running the Project

### 1. Install dependencies

Install the required Python libraries:

```bash
pip install -r requirements.txt
```

### 2. Generate sample audio (optional)

This creates synthetic horn, siren, and background noise samples.

```bash
python main.py --generate
```

### 3. Train the machine learning model

Train the classifier using the generated audio samples.

```bash
python main.py --train
```

### 4. Run the demo pipeline

This runs the full Vibedrive system including sound detection, classification, direction estimation, and alert generation.

```bash
python main.py --demo
```

### 5. Run the full pipeline (optional)

Run everything in sequence:

```bash
python main.py --all
```
